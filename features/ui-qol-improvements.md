# Quality of Life Improvements in the UI

There are a few loosely related changes that I would like you to make in the UI.

* the operations log display is truncating the project name unnecessarily in the first line
* we need a way to navigate to a project from an operations log entry -- proposal is to make the project name a link
* I need a way to copy the project ID to the clipboard from the project detail page. Perhaps a small icon next to the name and a
  tooltip.

I think that these can be done directly in the imbi-ui project without any additional API features.
